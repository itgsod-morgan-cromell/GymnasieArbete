import game.game
import game.gfx.animate

Projectile class(Entity)

    x, y, dir, type (which translates to image and other stuff), fly time, speed, damage

    projectile dictionary

    Update      --> self
    Render      --> self
    Type list   <-- type
    Direction   --> vector --> pos

    pos         <-- (x,y)

    Collision
    Particle effect

    Max fly time <-- int
    Fly time     --> int added per update cycle
    Travel speed -- Pixels --> int per update cycle
    Alive?      --> true or false
    Damage      --> How much damage projectile does on collision

    If not self.alive?
        self kill
